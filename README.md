# COVID-19 Data Analysis Dashboard

This project analyzes global COVID-19 data and presents insights through an interactive Power BI dashboard.

The goal of this project is to clean, transform, and visualize COVID-19 data to identify trends, regional impacts, and country level statistics.

---

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Data Transformation
4. Data Visualization using Power BI
5. Interactive Dashboard with slicers and KPIs

---

## Tools & Technologies

- Python (Pandas, NumPy)
- Jupyter Notebook
- Power BI
- CSV Dataset

---

## Data Cleaning

The dataset was cleaned using Python by performing:

- Handling missing values
- Removing duplicates
- Standardizing column names
- Converting data types
- Preparing dataset for visualization

---

## Dashboard Features

The Power BI dashboard includes:

### Global Overview
- Total Cases
- Total Deaths
- Death Rate
- Total Countries
- Total Regions

### Trend Analysis
- Yearly trend of Total Cases
- Death Rate trends across years

### Region Analysis
- Cases and deaths by WHO region
- Comparative analysis of regions

### Country Analysis
- Country level statistics
- Cases vs deaths comparison

### Top 10 Countries
- Countries with highest COVID cases

---

## Interactive Features

The dashboard contains slicers for:

- WHO Region
- Year
- Month
- Country

These filters allow dynamic exploration of the data.

---

## Key Insights

- The highest number of cases were reported in the United States.
- Significant variation exists between regions in terms of death rate.
- Cases increased rapidly during 2020–2022 before stabilizing.
